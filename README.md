# Google-Play-Store-Data-Analysis-Popular-App-Categories-Trends
An exploratory data analysis project on Google Play Store apps dataset, identifying popular categories, user ratings, installs, and trends. Includes data cleaning, visualization, and insights using Python, Pandas, Matplotlib, and Seaborn.

Google Play Store Data Analysis

This project performs **Exploratory Data Analysis (EDA)** on the Google Play Store dataset to identify trends in app categories, user ratings, installs, and popularity.

## Project Overview

The objective of this project is to analyze app data and uncover key insights such as:

- Most popular app categories  
- Relationship between ratings and installs  
- Trends in user engagement  
- Distribution of free vs paid apps  

## Dataset Description

The dataset contains information about apps available on the Google Play Store, including:

- App Name  
- Category  
- Rating  
- Reviews  
- Installs  
- Price  
- Type (Free/Paid)  
- Content Rating  

## Data Cleaning

The dataset required preprocessing before analysis:

- Removed missing and inconsistent values  
- Converted installs and price into numeric format  
- Cleaned special characters (+, $, commas)  
- Handled duplicates  
## Analysis Performed

### 1. Popular App Categories
- Identified categories with the highest number of apps  
- Compared category distribution  

### 2. Ratings Analysis
- Distribution of ratings  
- Categories with highest average ratings  

### 3. Installs Analysis
- Most downloaded categories  
- Relationship between installs and ratings  

### 4. Free vs Paid Apps
- Comparison of installs and ratings  
- Market dominance of free apps

## Visualizations

The project includes multiple visualizations:

- Bar charts (category popularity)  
- Histograms (ratings distribution)  
- Scatter plots (ratings vs installs)  
- Count plots (free vs paid apps)
